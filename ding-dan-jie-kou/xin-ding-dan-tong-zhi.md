### 未处理订单推送通知

---

有新订单推送通知门店主屏设备。

### 传输方式

---

websocket

### 返回值

---

| 字段 | 类型 | 说明 |
| :--- | :---: | :--- |
| task | string | 返回 |
| message | string | 请求返回状态信息 |
| data | int | 是否有新订单：1有，0无 |
| total | int | 待处理订单总数 |

* 有新订单，提示语音。

### 返回结果示例

---

```js
{
    "code": 0,
    "message": "请求成功",
    "data": 1,
    "total": 999
}
```
